# react-native-vulpes

React Native implementation of the Vulpes design system

## Installation

```sh
npm install react-native-vulpes
```

## Usage

```js
import Vulpes from "react-native-vulpes";

// ...

const result = await Vulpes.multiply(3, 7);
```
