# react-native-vulpes

React Native implementation of the Vulpes design system

## Installation

```sh
yarn add react-native-vulpes react-native-linear-gradient
```

## Usage

```js
import {H1} from "react-native-vulpes";

// ...

<H1 color="light_success">
```
